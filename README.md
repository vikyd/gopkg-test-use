# gopkg-test-use

Golang main package import https://github.com/vikyd/gopkg-test

# Usage

- download and unzip to anywhere:
  - https://github.com/vikyd/gopkg-test-use/archive/master.zip
- download dependency:
  - `go get gopkg.in/vikyd/gopkg-test.v2`
- run:
  - `go run main.go`
- should output:

```
Main
t01
```
